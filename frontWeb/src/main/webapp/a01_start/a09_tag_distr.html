AJAX란?

AJAX란 비동기적인 웹 애플리케이션의 제작을 위해 아래와 같은 조합을 이용하는 웹 개발 기법이다. 표현 정보를 위한 HTML (또는 XHTML) 과 CSS 동적인 화면 출력 및 표시 정보와의 상호작용을 위한 DOM, 자바스크립트, 웹 서버와 비동기적으로 데이터를 교환하고 조작하기 위한 XML, XSLT, XMLHttpRequest (Ajax 애플리케이션은 XML/XSLT 대신 미리 정의된 HTML이나 일반 텍스트, JSON, JSON-RPC를 이용할 수 있다). DHTML이나 LAMP와 같이 Ajax는 자체가 하나의 특정한 기술을 말하는 것이 아니며, 함께 사용하는 기술의 묶음을 지칭하는 용어이다.

실제로 AFLAC와 같이 사실상 Ajax에 바탕을 두고 있는 유사/복합 기술들이 속속 나타나고 있다. Ajax 애플리케이션은 실행을 위한 플랫폼으로 위에서 열거한 기술들을 지원하는 웹 브라우저를 이용한다. 이것을 지원하는 브라우저로는 모질라 파이어폭스, 인터넷 익스플로러, 오페라, 사파리, 구글 크롬 등이 있다. 단, 오페라는 현재 XSL 포맷팅 객체와 XSLT 변환을 지원하지 않는다.

제한적인 모델링과 매핑, 렌더링을 지원하는 툴이다. 전체적으로 기능이 오토데스크 3D 맥스, 오토데스크 마야, 시네마 4D에 비해서는 빈약한 편이지만, 자연 지형물을 묘사하는데 특화된 3D 툴 이다. 우수한 배치능력, 광원제어능력을 가지고 있어서 자연물을 묘사하는 능력에 있어서는 그들을 뛰어넘는다. 그리고 앞서 말한 소프트웨어들에 비해 익히기 쉽다. 강의나 지침서 없이 습득하는건 불가능에 가까운 그것들과 달리, 지침서를 보면서 배울 때보다 현저히 느리지만 혼자서 이것저것 만지는 걸로도 터득할 수 있다. 아바타의 배경 그래픽을 묘사하는 데 쓰인 프로그램이 바로 이 프로그램 이다.
